Applied ML

This final report was created using this click-through-rate dataset from kaggle. The data contains over 1,000,000 rows and 23 features which are a mix of numeric and categorical variables, including the target feature click: 0/1. We carry out systematic steps to preprocess the data - including missing value removal and imputation, standardization of variables and encoding of categorical features. After using various ML techniques to build models, we will utilize hyperparameter tuning to pick the optimal parameters and leverage several different evaluation metrics to measure the model performances in order to choose the most efficient one.
